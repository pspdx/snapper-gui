snapper-gui
===========

Dependencies
-----------

### ArchLinux:
	python3
	gtk3
	python-dbus
	python-gobject
	python-setuptools
	gtksourceview3
	snapper-git (AUR)
### openSUSE:
	python3
	dbus-1-python3
	python-gtksourceview-devel
	python3-setuptools
### Ubuntu
	python3
	libgtksourceview-3.0-1
	python3-dbus
	python3-setuptools

Installation
-----------

    $ git clone https://github.com/ricardo-vieira/snapper-gui/
    $ cd snapper-gui/
    # python3 setup.py install
